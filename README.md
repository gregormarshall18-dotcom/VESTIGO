# VESTIGO - Stocks Made Simple

A modern, responsive homepage for the VESTIGO financial app. Built with React, Vite, and Tailwind CSS.

## 🚀 Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Preview production build:**
   ```bash
   npm run preview
   ```

## 📁 Project Structure


## 🎨 Customization

### Colors
Edit `tailwind.config.js` to change the color scheme:
```javascript
colors: {
  primary: '#0066cc',    // Main blue
  secondary: '#00cc88',  // Accent green
}
```

### Content
Edit `src/App.jsx` to customize:
- Headlines and descriptions
- Features and benefits
- Pricing tiers
- Testimonials
- Navigation links

### Branding
- Replace the logo/emoji (currently 📈) with your brand image
- Update company name and taglines
- Customize footer links
